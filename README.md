## Curriculum Milestones

This repository is organized into distinct phases, each building upon the previous one to master the art of backend engineering.

### Milestone 1: The Core Foundation
*Explore the basics of handling requests and serving data.*
- **Objective**: Initialize a Node.js environment and set up a robust Express.js server.
- **Key Features**: Route handling, JSON responses, and environment configuration.
- **[Explore Phase 1](./week-1)**

### Milestone 2: Persistent Data Flow
*Mastering database integration and CRUD operations.*
- **Objective**: Transition from volatile memory to persistent storage with MongoDB.
- **Key Features**: 
  - Schema design with **Mongoose**.
  - Request validation using **Zod**.
  - Complete RESTful implementation (GET, POST, PUT, DELETE).
- **[Explore Phase 2](./week-2)**

### Milestone 3: Secure Industrial Architecture
*Scaling with security, middleware, and modular design.*
- **Objective**: Build a real-world Restaurant API with high-grade security.
- **Key Features**:
  - **Authentication & Authorization**: JWT-based security and Bcrypt password hashing.
  - **Modular Architecture**: Implementation of Controllers, Middlewares, and Routes.
  - **Advanced Database Patterns**: Complex relationships and data integrity.
- **[Explore Phase 3](./week-3)**

## Getting Started

To dive into any milestone, navigate to the respective directory and follow these steps:

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Set Up Environment**:
   Ensure you have a `.env` file configured with necessary variables (like `MONGO_URI` for later phases).

3. **Launch the Engine**:
   ```bash
   npm start
   # or for development
   npm run dev
   ```
